## 🚀 Featured Projects

### 🔹 Speech-to-Sign-Language
Built an AI-powered system that translates spoken language into sign language using wav2vec-based speech-to-text, NLP pipelines, MediaPipe Holistic tracking, and Blender animations for realistic sign output.

### 🔹 Deepfake Audio Detection using Autoencoders & wav2vec
Developed a deep learning pipeline to identify manipulated audio using Wav2Vec embeddings, noise reduction, spectrogram generation, and autoencoder-based anomaly detection.

### 🔹 Women Safety Analytics
Created a multi-model pipeline with person detection, gender classification, gesture recognition, anomaly detection, and geospatial heatmaps to identify high-risk situations and regions.

### 🔹 Tumor Detection
Worked on a medical imaging based project using machine learning / deep learning techniques for tumor detection and prediction.
